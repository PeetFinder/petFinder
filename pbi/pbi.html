OneDrive setup (recommended for auto Power BI refresh)
  1. Upload PetFinder_Tanauan_Batangas_Cleaned.xlsx to OneDrive (e.g. folder PetFinder).
  2. Azure Portal → App registrations → New app → API permissions → Files.ReadWrite.All (Application) → Grant admin consent.
  3. Create a client secret; copy Tenant ID, Client ID, and secret.
  4. In api/config.php, set onedrive.enabled to true and fill in your Microsoft email + file path.
  5. Power BI Desktop → change data source to the OneDrive Excel file → Publish.
  6. Power BI Service → semantic model → Schedule refresh every 15 minutes (no gateway needed).

Full steps: /api/pbi/onedrive-setup.php
